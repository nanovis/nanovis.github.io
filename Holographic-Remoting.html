OpenGL Holographic Remoting

Contact

Shorouq Altamimi, Ruwayda Alharbi

Description

Example demonstrates how one can use OpenGL to render to DirectX canvas within Hololens application. This allows users to easily adapt their OpenGL application and run them on the Hololens.
